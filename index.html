TaskPro: The best alternative to Task Manager

About

TaskPro is similar to Windows Task Manager, but it runs completely in PowerShell.

This app is useful when Task Manager is blocked, corrupted or otherwise.

Main Features

TaskPro has a simple interface, and a few main features:

Task List: A comprehensive list of currently running processes and other info, such as their process IDs and memory usage. It can be effortlessly refreshed with the "Refresh" button.

Stop/Kill task: Can either close processes or forcefully kill them with a simple text input. Simply input the name or PID of the process you want, and BAM, you can instantly stop or kill it.

Start task: Allows you to start an application given an executable file's URL. (Tip: By default, it starts in System32, so to start Explorer, for example, you can just type "explorer.exe".)
