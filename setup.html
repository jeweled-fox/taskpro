TaskPro Setup Guide

In 3 easy steps!

Step 1: Unzip the TaskPro.zip file

This can be done using the built-in system extractor or an app like WinRAR.

Step 2: Find the shortcut

You should find the shortcut to the app in the unzipped folder. If it doesn't work, find the tsk.ps1 file in the resources folder and make a shortcut from it.

Step 3: Open TaskPro

If it says there isn't an app to open it, find the powershell.exe app and select it to open the file. If you encounter a security error, dont worry, continue anyway. TaskPro is completely safe.
